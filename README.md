# 個人開発

## 目的
Next.js + TypeScriptの理解を深めるためにこの開発環境を作成。

## やったこと
- next-authでログイン機能実装

- APIからスプラトゥーン3のスケジュール情報を取得
- Google Analytics計測
- コンポーネントライブラリshadncn/uiを使用
